# 设计规范文档

> **版本**: v1.0.0  
> **最后更新**: 2024-11-18

## 1. 概述

本文档定义了AI能力交易平台的UI/UX设计规范，包括颜色系统、字体、间距、组件使用、交互设计等。

## 2. 设计原则

### 2.1 核心原则

- **一致性**: 保持设计元素的一致性
- **可用性**: 优先考虑用户体验
- **可访问性**: 支持无障碍访问
- **响应式**: 适配各种屏幕尺寸
- **性能**: 优化加载和交互性能

## 3. 颜色系统

### 3.1 主色调

```css
/* 品牌色 */
--primary: #2487FF;        /* 主蓝色 */
--primary-dark: #1A6FD9;   /* 深蓝色 */
--primary-light: #4DA3FF;  /* 浅蓝色 */

/* 辅助色 */
--accent: #FDE700;         /* 黄色强调 */
--accent-dark: #FDD700;    /* 深黄色 */

/* 中性色 */
--gray-50: #FAFAFA;
--gray-100: #F5F5F5;
--gray-200: #EEEEEE;
--gray-300: #E0E0E0;
--gray-400: #BDBDBD;
--gray-500: #9E9E9E;
--gray-600: #757575;
--gray-700: #616161;
--gray-800: #424242;
--gray-900: #212121;

/* 语义色 */
--success: #10B981;
--warning: #F59E0B;
--error: #EF4444;
--info: #3B82F6;
```

### 3.2 使用规范

- **主色**: 用于主要操作按钮、链接、重要信息
- **辅助色**: 用于强调、高亮、警告
- **中性色**: 用于文本、背景、边框
- **语义色**: 用于状态提示（成功、警告、错误）

## 5. 间距系统

### 5.1 基础间距

使用 4px 基准单位：

```css
--spacing-1: 0.25rem;  /* 4px */
--spacing-2: 0.5rem;   /* 8px */
--spacing-3: 0.75rem;  /* 12px */
--spacing-4: 1rem;     /* 16px */
--spacing-5: 1.25rem;  /* 20px */
--spacing-6: 1.5rem;   /* 24px */
--spacing-8: 2rem;     /* 32px */
--spacing-10: 2.5rem;  /* 40px */
--spacing-12: 3rem;    /* 48px */
--spacing-16: 4rem;    /* 64px */
--spacing-20: 5rem;    /* 80px */
```

### 5.2 使用规范

- **组件内部**: 使用小间距（4px, 8px, 12px）
- **组件之间**: 使用中间距（16px, 24px, 32px）
- **区块之间**: 使用大间距（48px, 64px, 80px）

## 8. 组件规范

### 8.1 按钮

#### 主要按钮
```tsx
<Button variant="default" size="default">
  主要操作
</Button>
```

#### 次要按钮
```tsx
<Button variant="outline" size="default">
  次要操作
</Button>
```

#### 幽灵按钮
```tsx
<Button variant="ghost" size="default">
  辅助操作
</Button>
```

#### 尺寸规范
- `sm`: 高度 32px，内边距 12px
- `default`: 高度 36px，内边距 16px
- `lg`: 高度 44px，内边距 20px

### 8.2 输入框

```tsx
<Input
  placeholder="请输入内容"
  className="h-9"
/>
```

### 8.3 卡片

```tsx
<Card className="rounded-xl border border-black/10 shadow-md">
  <CardHeader>
    <CardTitle>标题</CardTitle>
  </CardHeader>
  <CardContent>
    内容
  </CardContent>
</Card>
```

### 8.4 徽章

```tsx
<Badge variant="default">默认</Badge>
<Badge variant="secondary">次要</Badge>
<Badge variant="destructive">危险</Badge>
<Badge variant="outline">轮廓</Badge>
```

## 9. 布局规范

### 9.1 容器宽度

```css
--container-sm: 640px;
--container-md: 768px;
--container-lg: 1024px;
--container-xl: 1280px;
--container-2xl: 1400px;
```

### 9.2 网格系统

使用 Tailwind CSS Grid：

```tsx
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
  {/* 卡片 */}
</div>
```

### 9.3 间距规范

- **页面边距**: 16px (移动端) / 24px (桌面端)
- **区块间距**: 32px - 64px
- **组件间距**: 16px - 24px

## 10. 响应式设计

### 10.1 断点

```css
/* Tailwind CSS 默认断点 */
sm: 640px   /* 小屏设备 */
md: 768px   /* 平板 */
lg: 1024px  /* 桌面 */
xl: 1280px  /* 大桌面 */
2xl: 1536px /* 超大桌面 */
```

### 10.2 移动优先

使用移动优先的响应式设计：

```tsx
<div className="
  text-sm          /* 移动端 */
  md:text-base     /* 平板及以上 */
  lg:text-lg       /* 桌面及以上 */
">
```

## 11. 交互设计

### 11.1 悬停效果

```css
/* 按钮悬停 */
.button:hover {
  opacity: 0.9;
  transform: translateY(-1px);
  transition: all 0.2s;
}

/* 卡片悬停 */
.card:hover {
  box-shadow: var(--shadow-lg);
  transform: translateY(-2px);
  transition: all 0.3s;
}
```

### 11.2 过渡动画

```css
/* 标准过渡 */
transition: all 0.2s ease-in-out;

/* 快速过渡 */
transition: all 0.15s ease-out;

/* 慢速过渡 */
transition: all 0.3s ease-in-out;
```

### 11.3 加载状态

- 使用骨架屏（Skeleton）替代空白
- 使用加载动画（Spinner）
- 显示加载进度（Progress）

## 16. 无障碍设计

### 16.1 语义化HTML

使用正确的HTML标签：

```tsx
<button>按钮</button>  {/* 不是 <div> */}
<nav>导航</nav>        {/* 不是 <div> */}
```

### 16.2 ARIA属性

```tsx
<button aria-label="关闭对话框">
  <X />
</button>
```

### 16.3 键盘导航

- 所有交互元素支持键盘操作
- Tab顺序合理
- 焦点状态明显

### 16.4 颜色对比度

- 文本与背景对比度 ≥ 4.5:1
- 大文本对比度 ≥ 3:1

## 17. 性能优化

### 17.1 图片优化

- 使用 WebP 格式
- 懒加载图片
- 响应式图片（srcset）

### 17.2 代码分割

```tsx
const LazyComponent = lazy(() => import('./Component'));
```

### 17.3 动画性能

- 使用 `transform` 和 `opacity`
- 避免触发重排和重绘
- 使用 `will-change` 提示浏览器
